This directory contains results for Precision-Recall (PR) curve analysis performed using behavioral and textual features on YelpZip and YelpNYC datasets. PR curve analysis is performed under three settings i.e. reviewer, review and product-centric for three classification models SVM, MLP and LR. 

File names in this directory follow the following pattern:
DatasetName_SettingName_ClassificationModelName.png

Each file corresponds to a precision recall curve under a specific setting, classification model and dataset.
